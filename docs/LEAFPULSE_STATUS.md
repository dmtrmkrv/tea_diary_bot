# LeafPulse — статус продукта

**Обновлено: 19.07.2026** (следующее обновление — после очередного релиза; файл генерируется командой `/chat-brief` в Claude Code)

**LeafPulse** (leafpulse.ru) — веб-приложение «личный чайный дневник»: записи дегустаций, коллекция чая и посуды, отслеживание любимых вкусов. Публично запущен 29.06.2026, аудитория — русскоязычное чайное комьюнити.

## Продукт сейчас (что умеет)

- **Дегустации**: подробная запись по проливам (температура, время, ароматы, вкус, ощущения) и быстрая заметка за 20 секунд (теги-пресеты аромата/вкуса/ощущений, оценка, фото). Редактирование, фото-галерея, свои номера записей (у каждого пользователя своя нумерация — №1, №2…).
- **Коллекция**: сорта чая (с остатками в граммах, обложками, избранным, средним рейтингом) и посуда. Вкусовой профиль сорта собирается сам из тегов дегустаций.
- **Связки**: дегустация привязывается к сорту и посуде из коллекции; из карточки сорта видно все его дегустации и профиль вкуса.
- **Экспорт** дневника в CSV-таблицу.
- **Вход**: почта+пароль или Яндекс; перенос всех записей из старого Telegram-бота — в один клик на сайте. Онбординг для новичков, лендинг для незалогиненных.
- Сброс пароля по почте, удаление аккаунта, настройки профиля, тёмная/светлая тема.

## Новое за июль (материал для анонсов)

- Быстрая заметка на вебе (лёгкий способ фиксировать чай за 20 секунд).
- Вкусовой профиль сорта + избранные сорта + средний рейтинг в коллекции.
- Миниатюры фото — лента стала грузиться в ~10 раз быстрее.
- Персональные номера дегустаций в адресах страниц (красивые ссылки /tastings/42).
- Заголовки вкладок: имя чая в заголовке при шаринге ссылки.
- Единый стиль кнопок, полировка шторок коллекции.
- Плавный скролл по меню лендинга, починена навигация «назад».
- Онбординг с экраном переноса записей из бота.

## Цифры и каналы (внутреннее, не для публикаций)

- База: ~50 пользователей из эпохи бота + 4 новые регистрации с публичного запуска (все — по почте; Яндексом пока никто).
- Трафик пока минимальный: единичные новые посетители в день.
- **Работающий канал: тематические чайные TG-сообщества** — пост 18.07 дал конверсию лендинг→регистрация ~40% против ~6% у холодной аудитории. Формат повторять.
- QR-код на офлайн-карточке даёт редкие, но вовлечённые визиты.
- Воронка в Метрике сквозная: лендинг → клик CTA → форма → регистрация → первая запись. Слабое место сейчас — активация: новички регистрируются, но первую запись делают не сразу.
- **UTM для ссылок в постах**: `utm_source=<площадка: telegram|dzen|vk>`, `utm_medium=<social|qr>`, `utm_campaign=<имя публикации, напр. update_2026_07>`.
- Превью ссылки leafpulse.ru в Telegram не собирается (инфраструктурная особенность) — к постам прикладывать картинку вручную.

## Ближайшие планы (без публичных обещаний дат)

- Настройки cookie в профиле (отзыв согласия на аналитику).
- **Share-ссылки на дегустации** — публичная страница записи с красивым превью: главный кандидат на виральный канал.
- Активация новичков: довести от регистрации до первой заметки.
- Дальше по бэклогу: редактирование сортов/посуды, загрузка аватарки, статистика-дашборд, гид новичка.

## Не упоминать публично

- Telegram-бот как живой продукт (он сворачивается; «перенос записей из бота» упоминать можно и нужно).
- Точные цифры аудитории и регистраций.
- Любые детали безопасности и внутренней инфраструктуры.
